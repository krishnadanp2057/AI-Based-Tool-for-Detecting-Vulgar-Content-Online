About This Project

Welcome to the Offensive Language Checker web application! This project is designed to help users identify and filter offensive language in text submissions. Below are some key features and details about the project:

  • Purpose: The main goal of this application is to automatically detect and flag offensive language in user-submitted text. This is useful for moderating content and ensuring a positive environment in online platforms.
  • Features:
    • Text Checking: Users can submit text, which is then checked for offensive language using a pre-trained machine learning model.
    • Admin Dashboard: Admins can log in to view a list of offensive texts detected by the system. This feature helps in monitoring and managing flagged content.
    • User Authentication: The application supports user authentication with separate roles for regular users and admins, ensuring secure and appropriate access to features.
  • Technologies Used: This project uses Flask for the web framework, scikit-learn for the machine learning model, and basic HTML/CSS for the frontend. The machine learning model is trained to identify offensive language based on a dataset of labeled examples.
  • Future Improvements: Future enhancements could include a more sophisticated model, additional features for content management, and better user interface design.

Thank you for visiting the Offensive Language Checker web application. We hope you find it useful!
